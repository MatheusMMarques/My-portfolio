# Portfólio | Matheus Marques

Portfólio profissional desenvolvido em React para apresentar trajetória, experiência prática, cases, stack técnica e formas de contato.

## Objetivo

Reunir em uma única página a evolução profissional, a atuação em sistemas web, projetos aplicados, capacidades técnicas e canais de contato de Matheus Marques.

## Stack

- React
- JavaScript
- CSS
- EmailJS

## Como rodar localmente

```bash
npm install
npm start
```

O projeto será iniciado em modo de desenvolvimento.

## Status

Projeto em evolução.
